Welcome to ThingsBase. As far as we know this is the first hackerspace inventory management application. Hackerspace members often have limited resources (like space) but an abundance of hardware. To keep our space from becoming a complete tech graveyard, we decided to whip up and inventory application. We're working on it, so bear with us (or jump in!)

The source can be found on github at http://github.com/HackRVA/hackrva-inventory-application. If you have any Django experience (or not, I'm learning as I go, which might be a little obvious), don't be afraid to grab the source and get involved!

Happy Hacking!
